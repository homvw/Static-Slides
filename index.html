Präsentationen in HTML

Gliederrung

  1. Warum?
  2. Was?
  3. Wie?

Warum?

  • Alle Programme ähnlich zu PowerPoint sind „schwierig“
    • PowerPoint ist nicht Open Source und kostenintensiv.
    • Open-Source-Alternativen sind für meine Anforderungen nicht ausgelegt.
  • Das Setzen von Folien mit der LaTeX-Beamer-Klasse ist mühselig.
  • Ähnliches gilt für alle Pandoc Option ...
  • Reveal.js und ähnliche Frameworks sind mir zu klobig.

Was?

Ziel ist es, die Werkzeuge für statische Webseiten (HTML, CSS und ein wenig JS) zu nutzen, um damit Präsentationen erstellen zu können.

Vorteile: 

  • Plain Text
  • Software unabhängig
  • So flexibel wie möglich

Wie?

  1. Ein HTML-Dokument.
  2. Jede section entspricht einer neuen Folie.
  3. Die section Umgebungen werden bzgl. der z-Achse gestapelt, wobei nur die aktive sichtbar ist.
  4. Mittels CSS lässt sich die Präsentation frei gestalten.
  5. JS sorgt für nette Details wie
    • Pfeiltasten für die Navigation der Folien.
    • Automatische Foliennummerierung.
